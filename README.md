# Data Generation Project for Logistics Optimization

## Requirements

### Anaconda(recommend)

* Install Anaconda and Faker

```shell
$ conda install -c conda-forge faker
```

### Pure Python

* Python >= 3.10.x
* numpy, pandas, openpyxl, jupyterlab

```shell
$ pip install numpy pandas openpyxl faker jupyterlab 
```
